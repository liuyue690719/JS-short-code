Super Text Markup Language

Super Text Markup Language (HTML) is an application under the Standard Universal Markup Language (SGML), and it is also a specification, a standard, which marks all parts of the web page to be displayed by markup symbols. Page file itself is a text file, by adding markers in the text file, you can tell the browser how to display its content. Browsers read web pages in order, and then interpret and display the contents of their markers according to markers. Markers that write errors will not point out their errors, and do not stop their interpretation of the execution process.

Cascading Style Sheet

CSS is a language that defines style structures such as font, color, location, etc. It is used to describe the formatting and reality of information on Web pages. CSS styles can be stored directly in HTML pages or in separate style sheet files. Either way, style sheets contain rules for applying styles to elements of a specified type. When used externally, style sheet rules are placed in an external style sheet document with file extension _css

conclusion

Front-end development is the process of creating front-end interfaces such as Web pages or apps to present to users. User interface interaction of Internet products is realized through HTML, CSS and JavaScript, as well as various derived technologies, frameworks and solutions. It evolved from the production of web pages, and its name has obvious characteristics of the times. In the evolution of the Internet, web page production is the product of the Web 1.0 era. In the early days, the main content of websites was static, mainly pictures and text, and users'behavior of using websites was mainly browsing. With the development of Internet technology and the application of HTML5 and CSS3, modern web pages are more beautiful, interactive and powerful.
